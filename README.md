# Synchron Video

Простое веб-приложение для совместного просмотра видео в реальном времени.

## Возможности

- Комнаты для совместного просмотра (минимум 2 пользователя).
- Синхронизация **Play/Pause** между участниками комнаты.
- Синхронизация **перемотки** (seek).
- Поддержка ссылок:
  - Прямые ссылки на видео (например `.mp4`)
  - YouTube
  - Vimeo
- Ссылка-приглашение в комнату (`?room=<id>`), чтобы другу было проще подключиться через интернет.

## Запуск

```bash
npm install
npm start
```

После запуска откройте `http://localhost:3000` в двух вкладках/браузерах,
введите одинаковое название комнаты и смотрите видео синхронно.

## Как подключить друга через интернет

Нужно, чтобы ваш сервер был доступен по публичному URL.

### Вариант 1: быстрый временный доступ через туннель

1. Запустите приложение:

   ```bash
   npm start
   ```

2. Поднимите туннель (любой удобный сервис):

   ```bash
   cloudflared tunnel --url http://localhost:3000
   ```

3. Отправьте другу публичный URL из вывода `cloudflared`.
4. Зайдите в комнату и отправьте другу ссылку из поля **"Ссылка для друга"**.

### Вариант 2: постоянный доступ

- Разверните сервер на VPS/облаке.
- Настройте домен + HTTPS.
- Откройте доступ к приложению через Nginx/Caddy как reverse proxy.

## Ограничения

- Для внешних источников важны CORS/политики провайдера.
- Автовоспроизведение может блокироваться браузером до первого действия пользователя.
